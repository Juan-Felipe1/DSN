# Event Tester

Tests user interactions: clicks, hovers, focus, keystrokes, drag-drop, scrolls, combinations.

**Events covered:**
- Click, double-click, right-click
- Hover, focus, blur
- Keydown, keyup, enter, escape, tab
- Change (input, select)
- Submit, reset
- Scroll, resize, drag-drop
- Combinations (ctrl+click, shift+select)

**Tests:**
- Single events work correctly
- Rapid clicks handled (debouncing)
- Keyboard shortcuts work
- State changes after events
- Multiple simultaneous interactions

**Output:** PASS/FAIL per event scenario
