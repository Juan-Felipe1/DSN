# Visual Validator

Tests design consistency, layout, spacing, colors, typography, responsiveness.

**Expertise:** Dan Mall (design systems)

**Tests:**
- Color accuracy and contrast
- Spacing and alignment (grid adherence)
- Typography (font size, weight, line-height)
- Responsive behavior (mobile, tablet, desktop)
- Whitespace and visual hierarchy
- Icon consistency and sizing

**Output:** PASS/FAIL for each visual criterion, screenshots of failures
